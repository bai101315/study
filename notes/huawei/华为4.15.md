## SFT 模型复读问题分析
SFT 时没做 role 标记与 loss mask，把多轮对话整段算 loss，模型上线后容易**复读用户的话**。

### 根本原因
缺少角色边界与 mask，模型学到复述整段文本的模式。

### 其他干扰因素
- 训练轮数少：表现为「不会回答」
- tokenizer 能力不足：语义混乱
- 学习率过高：训练不稳定 / 发散

## 向量数据库 IVF（Inverted File）索引
IVF 通过 K-means 将所有向量聚类为 `nlist` 个中心，每个向量分配到最近的聚类。
查询逻辑：先找与 query 最近的 `nprobe` 个聚类中心，仅在这 `nprobe` 个聚类内精确搜索。
- `nprobe = nlist` 时：等价暴力搜索，100% 精度
- `nprobe` 越小：速度越快，但存在召回遗漏风险
- 经验平衡点：`nprobe = √nlist`
典型实现：FAISS 的 `IVFFlat`

## Hypothetical Questions（假设问题）增强索引
**原理**：让 LLM 为每个 chunk 生成 3-5 个该 chunk 可回答的问题，将问题与 chunk 共同索引/存入元数据。
**优势**：用户查询与假设问题匹配时，检索命中率远高于仅匹配原文文本，适配知识密集型文档。

## Step-back Prompting（Zheng et al. 2023）
示例问题：`X公司2023年Q3的营收是多少？`
执行逻辑：
1. 让 LLM 生成抽象高层问题：`X公司的财务指标分析方法是什么？`
2. 检索高层背景知识
3. 结合原始问题 + 检索知识，生成最终答案

**价值**：获取更全面背景知识，提升复杂推理任务回答质量。

## DDP 数据并行通信机制
**问题**：标准 DDP 训练中，每个 GPU 独立计算梯度后，保持模型权重一致需执行什么通信操作？
**答案**：各 GPU 独立算出梯度后，通过 **All-Reduce** 聚合梯度（求和/平均）并广播回所有设备，各卡再独立更新参数，保证模型权重完全一致。

## K-Means 核心知识点
1. 聚类前必须**预先指定簇的个数**
2. 迭代聚类核心准则：**样本点到各个簇质心的平方欧氏距离最小化**（最近邻原则）

## 最大内积搜索（MIPS）原理
MIPS 通过计算查询向量与文档向量的**点积（内积）**衡量相关性，**内积值越大，向量相似度越高**，系统召回得分最高的文档。

n 维向量：
$a = [a_1, a_2, ..., a_n]^T$
$b = [b_1, b_2, ..., b_n]^T$

内积公式：
$$a·b = a_1b_1 + a_2b_2 + \dots + a_nb_n$$

## $R^2$（决定系数）拟合优度
$R^2$ 是线性回归衡量**拟合优度**的核心指标。

1. 新增任意特征（含随机无关特征），$R^2$ **只会上升或不变，不会下降**
2. 含截距最小二乘模型训练集上，取值范围 $[0,1]$
3. $R^2=1$：完美拟合所有数据点
4. $R^2=0$：预测效果等价于仅用样本均值
5. $R^2$ 越接近 **1**，模型解释变异越多，拟合效果越好
6. **不同数据集的 $R^2$ 无直接可比性**

## 特征相关性方法对比
### 关键约束
1. 特征 X 与目标 Y 存在**复杂非线性、非单调相关性**（如 $Y=X^2$、$Y=\sin(X)$）
2. 特征 X **不服从正态分布**

### 各方法适配性
- **斯皮尔曼等级相关系数**：仅能捕捉**单调非线性关系**
- **方差分析 ANOVA F-value**：检验**分类特征**与连续目标的均值差异显著性
- **皮尔逊相关系数**：只衡量**线性相关**；要求变量近似正态分布
- **互信息 MI**：可衡量**任意依赖关系**（线性/非线性、单调/非单调）；**无分布假设**，适配非正态、复杂非线性场景

## 矩阵奇异值分解（SVD）
SVD 是通用**矩阵分解技术**，核心能力：**数据降维、特征提取、矩阵补全、噪声过滤**。

### 适用场景
1. **图像去噪**：SVD 可直接应用
2. **文本主题建模**：LSA/LSI 核心是对**词-文档共现矩阵**做 SVD，挖掘潜在语义
3. **推荐系统**：用户-电影高维稀疏评分矩阵，通过 SVD/Funk SVD 实现**矩阵补全、隐特征降维**，用于协同过滤

### 不适用场景
决策树：基于规则的非线性分类算法，训练预测**不涉及任何矩阵分解**。

## SmoothQuant 量化核心思想
大模型 INT8 量化精度损失主要来源：**激活值动态范围远大于权重**。

SmoothQuant 核心创新：
通过**数学等价线性变换**，将激活的大动态范围「平滑迁移」到权重侧，即 **activation scale 转移到权重**，平衡权重与激活的量化难度，降低 INT8 量化精度损失。

## 优化器特性区分
- Adam、AdamW、RMSprop：逐参数维护动量/二阶矩，**自适应**缩放学习率步长

## 设X和Y是两个随机变量，a、b和c是常数。以下关于期望性质的说法中
选项 A：\(E[XY] = E[X] \cdot E[Y]\) 总是成立：
这个等式仅当 X 和 Y 相互独立（或不相关）时成立，不是恒成立的。

\(Var(X) = E[X^2] - (E[X])^2\) 恒成立：
这是方差的定义式，对任何随机变量都成立，是概率论中最基础的公式之一。

选项 C：\(E[aX + bY + c] = aE[X] + bE[Y] + c\)
正确原因：这是期望的线性性质，是期望最核心、最重要的性质，无条件成立，与 X 和 Y 是否独立、是否相关没有任何关系

选项 D：\(E[X^2] \geq (E[X])^2\) 恒成立
正确原因：这是方差非负性的直接推论，也可以用 Jensen 不等式证明。

## 插值多项式核心性质

选项 A：插值多项式在节点处的函数值一定等于给定函数值
正确原因：这是插值的基本定义和核心要求，称为 "插值条件"。

选项 B：对于给定的 n+1 个互异节点，次数不超过 n 的插值多项式存在且唯一
正确原因：这是插值多项式的存在唯一性定理，是整个插值理论的基石。

选项 C：高次插值多项式通常能更好地逼近原函数
错误原因：这是插值理论中最经典的误区，高次插值会出现龙格现象（Runge Phenomenon），反而会导致精度急剧下降

选项 D：Lagrange 插值法和 Newton 插值法构造的插值多项式是相同的
正确原因：根据插值多项式的存在唯一性定理，满足相同插值条件的次数不超过 n 的多项式是唯一的。
