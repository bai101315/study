# 二分

## 库函数
查找范围: 左闭右开
```python
import bisect
# >=x
bisect.bisect_left(a, x, lo=0, hi=len(a))
# >x
bisect.bisect_right(a, x, lo=0, hi=len(a))
```
当找不到x的时候,返回第一个**大于x的元素索引**

| 情况        | bisect_left 返回值 | bisect_right 返回值 |
| :-------- | :-------------- | :--------------- |
| x < 所有元素  | 0               | 0                |
| x 在两个元素之间 | 第一个大于 x 的元素索引   | 第一个大于 x 的元素索引    |
| x 存在且无重复  | x 的索引           | x 的索引 + 1        |
| x 存在且有重复  | 第一个 x 的索引       | 最后一个 x 的索引 + 1   |
| x > 所有元素  | len(a)          | len(a)           |

## 自己实现
### 找某个元素
自己实现一般情况都是:
- 在一个区间里面,找**check()**函数,满足某种条件; 一般都采用闭区间
  
```python
def bian_search(self, nums: List[int], target: int, l, r) -> int:
    while l <= r:
        mid = l + r >> 1
        if nums[mid] == target:
            return mid
        elif nums[mid] > target:
            r = mid - 1
        else:
            l = mid + 1
    return -1
```
### 找区间

找“最后一个满足条件”的位置（右边界）； 不满足返回l-1, 可能是-1
```python
def rightmost_true(nums, cond, l, r):
    while l <= r:
        mid = (l + r) >> 1
        if cond(mid):     # 满足条件
            l = mid + 1   # 往右找
        else:
            r = mid - 1
    return r
```

找“第一个满足条件”的位置（左边界）：
```python
def leftmost_true(nums, cond, l, r):  
	while l <= r:  
		mid = (l + r) >> 1  
	if cond(mid): # 满足条件  
		r = mid - 1 # 往左收缩  
	else:  
		l = mid + 1  
	return l
```
