## 遍历二叉树
- 前序：中左右
- 中序：左中右
- 后序：左右中

## 构建二叉树
**关键：找到根结点，然后递归即可**
使用链表构建二叉树

- 构建node
```python
class TreeNode:
    __slots__ = ('val', 'left', 'right')
    def __init__(self, val):
        self.val = val
        self.left = None
        self.right = None
```
### 从前序和中序遍历构建二叉树

前序找根节点，左右定位
以前序为**基点**， 第一个元素是**根节点**，由此可以找到左右结点；进行递归
```python
def buildTree(self, preorder: List[int], inorder: List[int]) -> Optional[TreeNode]:

    if not preorder:
        return None

    root_node = preorder[0]
    left_size = inorder.index(root_node)
    # print(left_size, len(inorder))
    left = self.buildTree(preorder[1:1+left_size], inorder[:left_size])
    right = self.buildTree(preorder[1+left_size:], inorder[left_size+1:])
    return TreeNode(preorder[0], left, right)
```

### 从中序与后序遍历序列构造二叉树
跟上面差不多；根据后序遍历找到根节点，然后找到左右子树，递归建立
```python
def buildTree(self, inorder: List[int], postorder: List[int]) -> Optional[TreeNode]:
    # 以后序遍历为基点，找到左右子树
    if not postorder:
        return None

    val = postorder[-1]
    idx = inorder.index(val)

    left = self.buildTree(inorder[:idx], postorder[:idx])
    right = self.buildTree(inorder[idx+1:], postorder[idx:-1])
    return TreeNode(val, left, right)
```

### 根据前序和后序遍历构造二叉树
此时树不确定，可以进行假设：**规定preorder[1] 是 左子树的根**，那么左右子树就可以确定。就可以建树了


```python
def constructFromPrePost(self, preorder: List[int], postorder: List[int]) -> Optional[TreeNode]:
    if not preorder:
        return None
    if len(preorder) == 1:
        return TreeNode(preorder[0])
        
    left_root = preorder[1]
    left_idx = postorder.index(left_root)
    left_size = left_idx+1
    
    left = self.constructFromPrePost(preorder[1:1+left_size], postorder[:left_size])
    right = self.constructFromPrePost(preorder[1 + left_size:], postorder[left_size: -1])
    return TreeNode(preorder[0], left, right)

```


### 层序遍历构建二叉树

```python

def build_tree(arr):
    """根据层序遍历数组构建二叉树，None表示空节点"""
    if not arr:
        return None
    root = TreeNode(int(arr[0]))
    queue = [root]
    i = 1
    while queue and i < len(arr):
        node = queue.pop(0)  # 注意：pop(0)效率低，但n≤2e5可接受；可用deque
        # 左孩子
        if i < len(arr) and arr[i] != 'None':
            left = TreeNode(int(arr[i]))
            node.left = left
            queue.append(left)
        i += 1
        # 右孩子
        if i < len(arr) and arr[i] != 'None':
            right = TreeNode(int(arr[i]))
            node.right = right
            queue.append(right)
        i += 1
    return root
```
